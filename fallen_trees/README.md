### FALLEN TREES
**_Adds tree nodes to the falling_node group._**

**Version:** 1.2.2<br>
**License:** GPL v3.0

**Dependencies:** default (found in Minetest Game)<br>
**Supported:** mg, real_trees, desert_life

### Installation

Unzip the archive, rename the folder to fallen_trees and place it in<br>
../minetest/mods/

If you only want this to be used in a single world, place it in<br>
../minetest/worlds/WORLD_NAME/worldmods/

GNU+Linux - If you use a system-wide installation place it in<br>
~/.minetest/mods/

For further information or help see:<br>
https://wiki.minetest.net/Help:Installing_Mods
